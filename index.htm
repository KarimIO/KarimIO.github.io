Oak.js

An Online Instruction Set Simulator

Oak.js is an online Instruction Set Simulator, supporting assembly languages including MIPS and RISC-V. It is also designed as an Integrated Development Environment (IDE) for such languages, with features including theming, hotkeys, tabbed editing, syntax hilighting, number conversion, step-by-step debugging, register tracking, and more.
