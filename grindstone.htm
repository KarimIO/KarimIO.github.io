In-progress game engine developed by Karim Abdel Hamid. Progress started in May 2014, and has been reworked multiple times as I've learned more.

System Features

  • Extensible Cross-Platform system with support for Linux, Windows, with plans to support Mac, Mobile, and PS/Xbox
  • Multiple rendering APIs supported with Graphics abstraction layer. Currently OpenGL and Vulkan
  • Flexible Entity Component System allows for better extensibility, performance, and multithreading
  • Code with either the C++ native programming or our C# Scripting system
  • Reflection system exposes classes to the editor, save system, and networking
  • Integration with bullet physics
  • Frustum culling
  • Quick scene loading
  • Logging and Profiling systems

Rendering

  • Material and shader system allow for developers to easily create new materials in a cross platform way that hooks up to native systems
  • Viewport and camera systems allow for various different types of cameras to show the world both in game and in many times in editor
  • Deferred rendering to increase rendering performance
  • Post processing pipeline that includes Tonemapping, Bloom, SSAO, Automatic Exposure Adjustment, Color Grading, and more
  • Shadow mapping

Editor

  • Currently in the process of revamping the editor to be its own executable
  • Immediate-mode GUI for level editor, which includes an asset browser, component inspector, scene graph viewer, and multiple viewports
  • Some procedural editing features (perlin noise terrain, hydraulic weathering of terrain for improved realism)
  • Cubemap building
  • Image, Material, and Model conversion to compressed formats, both native and otherwise
  • Work on adding more tools such as material editor

Grindstone will be released for free for those developing free games, and licensable for others, at an undecided date.
